String Methods

  1. .length
    returns the length of the string

  2. .charAt()
    returns the character at the given index

  3. .charCodeAt()
    returns the UTF code of the character at a given index

  4. .at()
    new as of 2022 to improve .charAt(). Returns the character at the given index, but will also accept negative numbers, which represent indexes counted from the end

  5. []
    returns the character at the given index, but doesn't work to change anything about the string

  6. .slice()
    extract a portion of the string and store it in a new variable. Negative numbers count from end.
    • Arg 1: starting index;
    • Arg 2 (optional): end position (not inclusive)

  7. .substring()
    Similar to slice(), but interprets negative numbers as 0. Probably don't use it.

  8. .substr()
    Deprecated--don't use it. Similar to slice(), but the second parameter specifies length of the extrated part.

  9. .toUpperCase()
    Makes the string into UPPERCASE letters.

  10. .toLowerCase()
    Makes the string into lowercase letters.

  11. .concat()
  12. .trim()
  13. .trimStart()
  14. .trimEnd()
  15. .padStart()
  16. .padEnd()
  17. .repeat()
  18. .replace()
  19. .replaceAll()
  20. .split()
